

GUIDELINE HOW TO RUN THE CODE
==============================


1. Open wamp server and start its services

2. copy tram folder to www folder in wamp

3. Browse the PHPMyAdmin using "http://localhost/phpmyadmin

4. Create a new database and name it "tram_db"

5. Import the "tram_db.sql" file in this folder

6. Browse the system "thttp://localhost/tram/admin/" to access admin login 
 username: admin
 password: admin1234

7. Browse user's home page "http://localhost/tram/"
